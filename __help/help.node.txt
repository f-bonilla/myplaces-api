// __dirname
Por otro lado, cuando se utiliza __dirname como punto de partida para la ruta, siempre devuelve una ruta absoluta
basada en el directorio del módulo actual. Esto garantiza que la ruta resultante sea siempre la misma, independientemente
de dónde se ejecute el proceso node.
El uso de __dirname es especialmente útil cuando necesita hacer referencia a archivos o directorios que se encuentran en
el mismo directorio que el módulo actual. Mediante el uso de __dirname, puede asegurarse de que la ruta de acceso es
siempre correcta, independientemente de dónde se ejecute el proceso node.
