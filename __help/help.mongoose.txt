ValidationError:
Este error se produce cuando un documento no supera la validación, como cuando falta un campo obligatorio o un campo no supera una función de validación personalizada.
Código de error: undefined.

CastError:
este error se produce cuando un valor no se puede convertir al tipo esperado, como cuando se proporciona un identificador no válido.
Código de error: CastError.

MongoError:
Este error se produce cuando se produce un error específico de MongoDB, como un error de clave duplicada o un error de red.
Código de error: MongoError.

StrictModeError:
Este error se produce cuando un documento infringe la opción de esquema de modo estricto.
Código de error: StrictModeError.

VersionError:
Este error se produce cuando se guarda un documento con un número de versión obsoleto.
Código de error: VersionError.
