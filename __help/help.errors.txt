// Constructores de errores
Error
    Es la clase base para todos los errores en JavaScript. Puedes usarla para crear errores genéricos.
EvalError
    Es un error que se lanza cuando una llamada a la función eval() falla. Esta clase de error es obsoleta y generalmente no se usa en el código moderno de JavaScript.
RangeError
    Es un error que se lanza cuando un valor no está dentro del rango esperado.
ReferenceError
    Es un error que se lanza cuando se hace referencia a una variable que no ha sido definida.
SyntaxError
    Es un error que se lanza cuando el código JavaScript no es válido.
TypeError
    Es un error que se lanza cuando una operación intenta un tipo de dato que no es válido.
URIError
    Es un error que se lanza cuando una función global URI no puede completar una operación.
-- ademas en mozilla habla de estos
EvalError
    Crea una instancia que representa un error que ocurre con respecto a la función global eval().
InternalError
    Crea una instancia que representa un error que ocurre cuando se produce un error interno en el motor de JavaScript. Por ejemplo: "demasiada recursividad".

Es importante mencionar que aunque puedes usar estos constructores de errores para crear diferentes tipos de errores,
también puedes crear tus propios errores personalizados extendiendo la clase Error incorporada de JavaScript, ejem:
new Error("MyErrorType")
